# 🎯 Zam Wesh - Inspire, Motive et Progresse Chaque Jour

Un site web moderne et professionnel dédié à l'inspiration, la motivation et le développement personnel.

## ✨ Caractéristiques

### 🎨 Design Moderne
- **Palette de couleurs** : Noir, Blanc et Bleu
- **Responsive** : Adapté pour tous les appareils (mobile, tablette, desktop)
- **Animations élégantes** : Transitions fluides et effets visuels captivants
- **Interface intuitive** : Navigation claire et facile à utiliser

### 📱 Sections du Site

1. **Accueil (Hero)**
   - Logo et nom du site
   - Photo de profil ronde avec animation
   - Slogan inspirant
   - Boutons d'appel à l'action

2. **À Propos**
   - Description de la mission
   - Liste des avantages avec icônes
   - Contenu inspirant

3. **Conseils et Motivation**
   - 6 cartes avec conseils pratiques
   - Icônes emoji attrayantes
   - Animations au survol
   - Thèmes : Discipline, Objectifs, Mentalité, Gestion du temps, Communauté, Persévérance

4. **Blog**
   - Articles de blog avec images
   - Dates de publication
   - Liens "Lire plus"
   - Gradients de couleurs attrayants

5. **Contact**
   - Formulaire de contact complet
   - Informations de contact
   - Validation des données
   - Notifications de succès/erreur

6. **Footer**
   - Droits d'auteur
   - Liens de réseaux sociaux

## 🛠️ Technologies Utilisées

- **HTML5** : Structure sémantique
- **CSS3** : Styles modernes avec variables CSS, Flexbox, Grid
- **JavaScript** : Interactivité, animations, gestion des formulaires

## 🚀 Fonctionnalités JavaScript

- **Menu mobile responsif** avec hamburger animé
- **Formulaire de contact** avec validation email
- **Notifications** en temps réel (succès/erreur)
- **Intersection Observer** pour les animations au scroll
- **Scroll spy** pour la navigation active
- **Smooth scrolling** pour les ancres
- **Effets parallaxe** sur le hero
- **Menu mobile** qui se ferme au clic sur un lien

## 🔧 Personnalisation

### Modifier les couleurs
Éditez les variables CSS dans `style.css` :
```css
:root {
    --primary-color: #0066ff;      /* Couleur principale (bleu) */
    --secondary-color: #1a1a1a;    /* Couleur secondaire (noir) */
    --accent-color: #ffffff;        /* Couleur d'accent (blanc) */
    --text-color: #333333;          /* Couleur du texte */
    --light-bg: #f8f9fa;            /* Fond clair */
}
```

### Ajouter votre photo de profil
Remplacez l'URL du placeholder par votre image :
```html
<img src="votre-image.jpg" alt="Profil" class="profile-pic">
```

### Modifier le contenu
Éditez directement le texte dans `index.html` :
- Titres des sections
- Descriptions
- Cartes de conseils
- Articles de blog
